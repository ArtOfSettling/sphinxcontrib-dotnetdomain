.NET Domain Syntax
==================

Below are the supported syntax rules to the .NET domain:

Generic Types
=============

Generic type supported syntax::

    .. dn:class:: Foobar<T>
    .. dn:class:: Foobar<T,T>
    .. dn:class:: Foobar<TFoo,TBar>
    .. dn:class:: Foobar<T,<string>>
    .. dn:class:: Foobar<T,<T,<string>>>
